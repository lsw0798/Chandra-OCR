## Chandra OCR
문서를 HTML/Markdown/JSON 세가지 구조로 변환
vLLM 모드/Huggingface 로컬 모드가 있음

* 이미지 데이터 외부 전송을 방지하기 위해 로컬에서 진행
* pdf가 아닌 이미지 형식은 HTML 구조로 밖에 변환이 안됨

### OCR(Optical Character Recognition): 광학문자인식
이미지나 pdf 문서에서 텍스트 데이터를 인식해서 추출하는 기술

### 작동 방식
1. 문서를 흑백 버전으로 변환 후 스캔한 이미지에서 밝은 영역과 어두운 역역 분석

2. 디지털 이미지 정리 - 외부 픽셀 제거, 기울기 보정, 인쇄된 이미지의 일부인 그래픽 규칙 및 상자 제거, 스크립트 텍스트 포함 여부 결정 등

3. 텍스트 인식: 어두운 영역은 문자, 숫자, 기호를 찾기 위해 처리 - 일반적으로 한 번에 하나의 문자, 단어 또는 텍스트 블록을 대상. -> 문자는 **패턴 인식** 또는 **기능 인식**의 두 가지 알고리즘 중 하나를 사용하여 식별

4. 패턴 인식(또는 패턴 매칭): 스캔한 문서/이미지 템플릿과 비교하여 문자를 인식할 수 있도록 다양한 글꼴과 형식의 텍스트 예시에 대해 학습됨.
글리프 - 모양, 크기 및 글꼴의 고유한 각 조합.
이 기능이 작동하려면 문자가 OCR 프로그램에서 이미 학습된 글꼴로 되어 있어야 함.

5. 기능 인식(감지 또는 추출): OCR 프로그램이 학습되지 않은 글꼴을 분석할 때 사용.
OCR은 스캔한 문서의 문자를 인식하기 위해 특정 문자 또는 숫자의 기능에 관한 규칙을 적용.
문자가 식별되면 아스키 코드로 변환
레이아웃(문서 이미지의 구조) 인식: 페이지를 텍스트 블록, 표 또는 이미지와 같은 요소로 나눈다.

6. 후처리: 수집된 정보는 편집 가능한 형식 또는 PDF인 디지털 파일로 저장
